# B-UI-4 modal.opened / Wait ForModal probe
# Version: 1.0 · 2026-04-23
# Requires: VS, AK
# Purpose: Exercise Wait ForModal at both reviewer and approver
#          modal-render checkpoints from dual_session_test v1.4.
#          Retains the Cadence iframe form-render Pause (out of
#          B-UI-4 scope; subject of compass_form_ready brief).
#
# Bootstrap: derived from dual_session_test v1.4. Form Submit
# requires Compass loaded + MY REQUESTS / BROWSE selected, so
# the script bootstraps both sessions before the probe-specific
# Wait ForModal calls.

# — preflight · presence —————————————————————————————————
Assert session VS is connected
Assert session AK is connected

# — preflight · navigate both sessions to Compass ————————
VS: Set View "compass"
AK: Set View "compass"
Wait ForLocation VS "compass.my_work" timeout=30000
Wait ForLocation AK "compass.my_work" timeout=30000
VS: Set Tab "MY WORK"
AK: Set Tab "MY WORK"

# — VS submits ———————————————————————————————————————————
VS: Set Tab "MY REQUESTS"
VS: Set SubTab "BROWSE"
VS: Form Open "Expense Report"
Pause Form open — confirm Expense Report modal is visible on Vaughn's screen
# (Cadence iframe — out of B-UI-4 scope; awaits compass_form_ready brief)
VS: Form Insert "Employee Name" "Vaughn Staples"
VS: Form Insert "Trip Start Date" "2026-04-14"
VS: Form Insert "Trip End Date" "2026-04-14"
VS: Form Select "Business Purpose" "client"
Wait 500
VS: Form Insert "Purpose Description" "B-UI-4 modal.opened probe"
VS: Form Insert "Customer Name" "Apex Consulting Group"
VS: Form Insert "Customer Location" "Phoenix, AZ"
Wait 500
VS: Form Submit
Wait ForEvent "form.submitted" → $instance_id
Wait ForInstance $instance_id for launched
Log "Package live · $instance_id"

# — step 1: VS reviews · Wait ForModal (reviewer) ————————
VS: Set Tab "MY WORK"
Wait ForRoute $instance_id to VS
Wait ForQueueRow $instance_id to VS
VS: Click ForInstance $instance_id "Review"
VS: Wait ForModal "Document Review Request" for reviewer
Log "✓ B-UI-4 · Reviewer modal opened on VS"
VS: Click "Approve"
Wait ForRoute $instance_id to AK
Log "Step 1 complete"

# — step 2: AK approves · Wait ForModal (approver) ———————
AK: Set Tab "MY TIME"
AK: Set Tab "MY WORK"
Wait ForQueueRow $instance_id to AK
AK: Click ForInstance $instance_id "Approve"
AK: Wait ForModal "Document Review Request" for approver
Log "✓ B-UI-4 · Approver modal opened on AK"
AK: Click "Approve"
Wait ForInstance $instance_id for completed timeout=30000
Log "Step 2 complete · AK approved"

Log "✓ B-UI-4 probe passed · $instance_id"
